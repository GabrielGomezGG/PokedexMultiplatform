# PokedexMultiplatform
## Proyecto Kotlin Multiplatform para la materia Taller de Programación utilizando las siguientes tecnologías:
- [Kotlin Multiplatform](https://kotlinlang.org/docs/multiplatform.html)
- [Android Studio](https://developer.android.com/studio)
- [Coroutines](https://kotlinlang.org/docs/coroutines-overview.html)
- [Kotlin Serialization](https://kotlinlang.org/docs/serialization.html)
- [Ktor Client](https://ktor.io/docs/getting-started-ktor-client-multiplatform-mobile.html)
- [SQLDelight](https://cashapp.github.io/sqldelight/2.0.0/js_sqlite/multiplatform/)
